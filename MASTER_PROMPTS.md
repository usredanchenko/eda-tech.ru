# 🎯 Мастер-промпты для доработки EDA-TECH

## 🚀 Prompt #1: Полная локализация и русификация
**Роль:** Senior i18n Specialist + Frontend Developer
**Приоритет:** 🔴 Критический
**Время:** 2-3 дня

### Контекст
Сайт EDA-TECH частично переведен на русский. В файле `useLanguage.tsx` есть переводы, но не все компоненты их используют. Обнаружены hardcoded английские тексты.

### Техническое задание
```
РОЛЬ: Ты опытный i18n-специалист и React-разработчик

ЗАДАЧА: Завершить полную локализацию сайта EDA-TECH на русский язык

ТЕКУЩЕЕ СОСТОЯНИЕ:
- Система переводов уже есть (useLanguage hook)
- Переводы хранятся в translations объекте
- Частично внедрено в компоненты (60-70%)

ПРОБЛЕМЫ:
1. Contact.tsx - hardcoded тексты: "Get In Touch", "Send us a message", "Reach us directly"
2. Footer.tsx - hardcoded тексты в quickLinks и services
3. History.tsx - частично переведен timeline 
4. OrderForm.tsx - некоторые поля формы не используют переводы

ПЛАН ДЕЙСТВИЙ:
1. Проанализировать все компоненты и найти hardcoded тексты
2. Добавить недостающие ключи в translations объект
3. Заменить все hardcoded тексты на t('key') вызовы
4. Проверить качество русских переводов
5. Убедиться что переключение языка работает везде

ТРЕБОВАНИЯ:
- Сохранить существующую архитектуру
- Использовать существующий useLanguage hook
- Поддержать fallback на английский для отсутствующих ключей
- Русские тексты должны звучать естественно, не как перевод

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
✅ 100% локализованный интерфейс
✅ Все тексты переключаются между EN/RU
✅ Качественные русские переводы
✅ Единая система управления переводами
```

## 🎨 Prompt #2: Красивые шрифты для кириллицы
**Роль:** UI Designer + Frontend Developer
**Приоритет:** 🔴 Критический  
**Время:** 1-2 дня

### Контекст
Сейчас используются системные шрифты, которые выглядят скучно для кириллицы. Нужны современные Google Fonts с хорошей поддержкой русских символов.

### Техническое задание
```
РОЛЬ: Ты UI-дизайнер со специализацией на типографике + Frontend-разработчик

ЗАДАЧА: Внедрить красивые современные шрифты для кириллицы

ТЕКУЩЕЕ СОСТОЯНИЕ:
- Используются системные шрифты: ui-sans-serif, system-ui
- В globals.css настроены CSS переменные для шрифтов
- TailwindCSS конфигурация поддерживает кастомные шрифты

ПРОБЛЕМА: 
Стандартные шрифты выглядят скучно и не стильно для кириллических текстов

ТРЕБОВАНИЯ К ШРИФТАМ:
- Отличная поддержка кириллицы
- Современный, стильный вид
- Хорошая читаемость на всех размерах
- Подходит для темной темы сайта
- Variable fonts для лучшей производительности

РЕКОМЕНДУЕМЫЕ ШРИФТЫ:
1. Inter - универсальный, очень популярный
2. Manrope - современный, стильный
3. Onest - российский шрифт, отлично для кириллицы
4. Commissioner - уникальный, выразительный
5. Golos Text - специально для русского текста

ПЛАН РЕАЛИЗАЦИИ:
1. Выбрать 1-2 шрифта (основной + акцентный)
2. Подключить через Google Fonts CDN
3. Настроить font-display: swap для быстрой загрузки
4. Обновить CSS переменные в globals.css
5. Проверить на всех экранах и размерах текста
6. Оптимизировать line-height для кириллицы

ФАЙЛЫ ДЛЯ ИЗМЕНЕНИЯ:
- index.html - добавить <link> для Google Fonts
- globals.css - обновить --font-* переменные
- возможно tailwind.config.js

ВАЖНО:
- НЕ ломать существующий дизайн
- Сохранить иерархию размеров текста
- Проверить на мобильных устройствах
- Учесть performance (загружать только нужные веса)

РЕЗУЛЬТАТ:
✅ Красивая современная кириллица
✅ Стильный внешний вид текстов
✅ Сохранена производительность
✅ Кроссплатформенная совместимость
```

1. VITE КОНФИГУРАЦИЯ:
```js
