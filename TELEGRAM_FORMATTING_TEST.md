## 🛠 УСТРАНЕНИЕ ПРОБЛЕМ

### Сообщение не приходит:
```bash
# Проверить логи бота
sudo pm2 logs eda-tech-telegram-bot

# Проверить статус
sudo pm2 status

# Проверить webhook
curl "https://api.telegram.org/bot8346452116:AAFq5oOG_e-zF4A1Tmw_faWlP2WTspxw1Jk/getWebhookInfo"
```

### Неправильное форматирование:
- Проверь что Markdown поддерживается (`parse_mode: 'Markdown'`)
- Убедись что специальные символы экранированы
- Проверь длину сообщения (максимум 4096 символов)
